왜 AllDoc인가

Spotlight가 못 찾는 것까지, 내가 정한 폴더 안에서 정확하게.

⚡

본문 전문검색 (FTS5)

SQLite FTS5로 문서 본문을 인덱싱. 수만 개 문서도 즉시 검색하고 관련도순으로 정렬, 일치 부분을 스니펫으로 보여줍니다.

🔍

이름 + 본문 동시에

이름·본문을 함께 또는 따로 검색. *.md 같은 글롭 패턴, 한글 부분검색도 지원합니다.

📂

내가 정한 폴더만

관리할 폴더를 직접 지정. 전체 폴더·즐겨찾기·개별 폴더를 사이드바에서 오가며 봅니다.

👁

Finder식 미리보기

Space로 Quick Look, 미리보기 중 ↑/↓로 다음 문서로 넘기기 — Finder와 똑같이.

🚀

대용량도 즉시

폴더 목록 디스크 캐싱 + 백그라운드 증분 색인으로 3만 개 넘는 폴더도 끊김 없이 열립니다.

🧩

가벼운 네이티브

SwiftUI + AppKit, 내장 SQLite 사용으로 외부 런타임 의존성 최소화. Xcode 없이 빌드됩니다.
